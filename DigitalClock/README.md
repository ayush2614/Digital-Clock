# Digital Clock (C++)

A console-based digital clock application written in C++.

## ✨ Features
- Real-time HH:MM:SS clock
- Shows current date (Day, Month DD, YYYY)
- ASCII border design
- Cross-platform (Windows, Linux, macOS)
- Graceful exit with Ctrl+C

---

## ⚙️ Build & Run

### Using Makefile (Recommended)
```bash
make
./digital_clock
